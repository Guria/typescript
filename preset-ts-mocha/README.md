# Northbrook TypeScript Mocha Preset

All your TypeScript and Mocha plugins in one conveninent place!

Contains the following plugins

- @northbrook/ts-build
- @northbrook/ts-create
- @northbrook/ts-init
- @northbrook/tslint
- @northbrook/ts-mocha

## Get it

```
npm install -D @northbrook/preset-ts-mocha
```

## Use it

```
{
  "packages": [],
  "plugins": [
    "preset-ts-mocha"
  ]
}
```
